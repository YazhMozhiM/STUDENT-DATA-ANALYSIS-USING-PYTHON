## STUDENT-DATA-ANALYSIS-USING-PYTHON

# Overview
This project analyzes student performance data using Python. The dataset is processed to calculate key statistics, identify trends, and understand the relationship between study hours and academic performance.

## Features
- Load and process student data from a CSV file
- Calculate average marks and study hours
- Identify the top-performing student
- Analyze the correlation between study hours and marks
- Visualize data using graphs

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib

## Dataset
The dataset contains the following information:
- Student Name
- Study Hours
- Attendance
- Marks
